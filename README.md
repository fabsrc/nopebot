# Nopebot

> Nope. 

*A Twitter Bot.*

## Install

```bash
$ npm install
```

## Start

```bash
$ npm start
```

Nodebot requires `CONSUMER_KEY`, `CONSUMER_SECRET`, `ACCESS_TOKEN` and `ACCESS_TOKEN_SECRET` environment variables for Twitter. They can be specified in a `.env` file.

## Test

```bash
$ npm test
```

## Develop

```bash
$ npm run dev
```
